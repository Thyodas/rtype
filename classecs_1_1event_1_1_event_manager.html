Detailed Description

Manages event handling and dispatching in the ECS system.

This class is responsible for registering event listeners, emitting events, and managing the event dispatch process. It allows for loose coupling between different components of the ECS system by using an event-driven approach.

Dispatches all events in the event queue to their respective listeners.

This method goes through the event queue, dispatching each event to the registered listeners for its type. It should be called regularly to ensure timely handling of events.
